# Beanstalk

Beanstalk is a Discord bot for Android: Netrunner. I wrote a little about it
[here][blog].

### Screenshots

![Heimdall](/assets/heimdall.png)

### Installation

```
git clone git@github.com:dougblack/beanstalk.git
cd beanstalk
make install
```

### Usage:

```
BEANSTALK_TOKEN=<bot_token> python beanstalk/bot.py
```
