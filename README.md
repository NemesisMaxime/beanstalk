# Beanstalk

Beanstalk is a Discord bot for Android: Netrunner.

![Professor](/assets/professor.png)

### Installation

```
git clone git@github.com:dougblack/beanstalk.git
cd beanstalk
make install
```

### Usage:

```
BEANSTALK_TOKEN=<bot_token> python beanstalk/bot.py
```
